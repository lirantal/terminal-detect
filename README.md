# About

`terminal-detect`is a Node.js CLI to help in troubleshooting a terminal user interface support for visual properties like support for unicode chars, colors, and so on.

# Usage

```bash
npx terminal-detect
```

# Install

```bash
npm install -g terminal-detect
```
